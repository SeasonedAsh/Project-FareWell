```markdown
# Contributor Covenant Code of Conduct

## Vårt løfte
Vi forplikter oss til å gjøre fellesskapet åpent, trygt og inkluderende.

## Våre standarder
- Vær respektfull i språk og handlinger.
- Aksepter konstruktiv kritikk.
- Vis empati mot andre bidragsytere.

## Uakseptabel oppførsel
- Trakassering, diskriminering eller personlig angrep.
- Deling av andres private informasjon uten samtykke.
- Andre handlinger som skader fellesskapet.

## Håndheving
Rapporter til: conduct@lexinav.no  
Prosjekt-teamet følger opp raskt og konfidensielt.

Basert på [Contributor Covenant](https://www.contributor-covenant.org/)
```
