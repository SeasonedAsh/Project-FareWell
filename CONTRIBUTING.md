# Contributing to LexiNAV

Takk for at du vurderer å bidra 🚀  
LexiNAV er et åpent prosjekt under **Project-FareWell**.

## Hvordan bidra
- **Issues** → Bruk GitHub Issues.
- **Pull Requests** → Lag branch, gjør endringer, test lokalt, åpne PR.

## Pull Request Guidelines
1. `git checkout -b feature/my-feature`
2. Skriv klare commit-meldinger.
3. Oppdater tester/docs.
4. Åpne PR og beskriv endringene.

## Coding Style
- TypeScript/React: eslint + prettier.
- API: Fastify/Prisma med `neverthrow` + `zod`.
- n8n: flows alltid eksportert til JSON.
- “Files are truth”: eksporter schema/flows/pages til Git.

## Community Guidelines
- Vær respektfull, inkluderende og konstruktiv.
