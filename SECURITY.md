```markdown
# Security Policy

## Supported Versions
Vi støtter siste release (0.x-serien).  
Eldre versjoner får kun sikkerhetsfixer ved behov.

## Reporting a Vulnerability
1. Ikke opprett offentlig issue.  
2. Send privat e-post til security@lexinav.no eller bruk GitHub Security Advisories.  
3. Beskriv feilen, reproduksjon, og risiko.

Vi lover:
- Bekreftelse innen 48 timer.
- Tidslinje for fix.
- Oppdatering til fix er ute.
- Kreditering (hvis ønskelig).
```
